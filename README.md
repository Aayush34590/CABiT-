# 🚖 CABiT+ Smart Cab Booking System

## 📌 Problem Statement

Urban transportation systems often lack safety features, efficient fare calculation, and environmental awareness. This leads to unsafe and inefficient travel.

---

## 💡 Solution

CABiT+ is a Java-based smart cab booking system that allows users to book rides, calculate fares, store ride data, and trigger emergency alerts.

---

## ✨ Features

* 🚖 Book Cab Ride
* 📊 Fare Calculation (Base fare + distance)
* 📁 File Saving (rides.txt)
* 🆘 SOS Emergency Feature
* 🌱 Eco Tracking (Fuel Usage)
* 📋 View All Rides

---

## 🛠️ Technologies Used

* Java
* OOP (Classes & Objects)
* File Handling
* GitHub

---

## ▶️ How to Run

1. Open Command Prompt

2. Go to project folder:
   cd Desktop\CABiT+\src

3. Compile:
   javac *.java

4. Run:
   java Main

---

## 📸 Output

* User enters details
* Ride gets booked
* Fare calculated
* Data saved in file
* SOS alert works
