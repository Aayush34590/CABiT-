 CABiT+ Smart Cab Booking System
 Problem Statement
In the current context of urban cities, it is observed that transportation systems are usually not equipped with proper safety features, fare estimation, and even environmental awareness. This may cause a problem in terms of efficiency in travel and safety concerns.

 Solution
CABiT+ is a simple yet powerful smart cab booking system developed by using Java. This project provides a facility for booking a ride, calculating the fare, saving ride information, and even sending SOS alerts in case of emergencies. Eco tracking is also included in this project, which is more realistic.

 Features
 Book a cab ride easily by using this project
 Fare calculation is automatically done by this project
 Save ride information in a file named rides.txt
 SOS alert is included in this project for safety concerns
 Eco tracking is included in this project for a more realistic purpose
 Display all previously booked rides
 Technologies Used
Java
Object-Oriented Programming
File Handling
GitHub
 How to Run
Open Command Prompt
Go to the project location: cd Desktop\CABiT+\src
Compile the program: javac *.java
Run the application: java Main
