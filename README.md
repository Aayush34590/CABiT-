#  CABiT+ Smart Cab Booking System

##  Problem Statement

In today's urban environment, transportation systems often lack proper safety features, accurate fare estimation, and awareness of environmental impact. This can lead to inefficient travel and safety concerns for passengers.

---

##  Solution

CABiT+ is a simple and effective smart cab booking system developed using Java. It allows users to book rides, automatically calculate fares, store ride details, and trigger emergency alerts when needed. The project also includes eco-tracking to estimate fuel usage, making it more practical and meaningful.

---

##  Features

*  Book cab rides easily through a menu-driven interface
*  Automatic fare calculation based on distance
*  Store ride details in a file (`rides.txt`)
*  SOS emergency alert feature for safety
*  Eco tracking to estimate fuel consumption
*  View all previously booked rides

---

##  Technologies Used

* Java
* Object-Oriented Programming (OOP)
* File Handling
* GitHub for version control

---

##  How to Run

1. Open Command Prompt

2. Navigate to the project folder:
   `cd Desktop\CABiT+\src`

3. Compile the program:
   `javac *.java`

4. Run the application:
   `java Main`

---

##  Output

* User enters name and distance
* Ride is booked successfully
* Fare is calculated automatically
* Ride details are saved in a file
* SOS alert can be triggered in an emergency
